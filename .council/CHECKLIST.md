# Shipping Checklist

## Before coding
- [ ] BRIEF updated
- [ ] Code Map generated (CODEMAP.md)
- [ ] Task has acceptance criteria

## Before commit
- [ ] Tests added/updated
- [ ] All checks pass locally
- [ ] NOTES updated
- [ ] If architecture changed: DECISIONS entry added

## Before merge
- [ ] PR description includes plan + verification evidence
- [ ] Rollback plan clear
